WASM binary

In this sample OpaDotNet will use policy compiled as WASM binary.

1. Create policy

Create policy file quickstart/policy.rego with the following contents:

package example

default hello = false

hello {
    x := input.message
    x == data.world
}

2. Compile policy module

In this example we will compile policy manually, see Compilation section for more details.

opa build -t wasm -e example/hello quickstart/example.rego

opa CLI will produce bundle.tar.gz file.

Extract policy.wasm from bundle bundle.tar.gz:

tar -zxvf bundle.tar.gz /policy.wasm

3. Add usings


using OpaDotNet.Wasm;

4. The code


// Create evaluator from compiled policy module.
using var engine = OpaEvaluatorFactory.CreateFromWasm(File.OpenRead("data/policy.wasm"));

// Set external data.
var data = """{ "world": "world" }""";
engine.SetDataFromRawJson(data);

// Evaluate. Policy query will return false.
var deny = engine.EvaluatePredicate(new { message = "hi" });

if (deny.Result)
{
    // Should not get here.
}
else
{
    // Wrong password.
}

// Evaluate. Policy query will return true.
var approve = engine.EvaluatePredicate(new { message = "world" });

if (approve.Result)
{
    // Correct password.
}
else
{
    // Should not get here.
}
